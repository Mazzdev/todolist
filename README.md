# ToDoList – projekt Android

Projekt zaliczeniowy z przedmiotu **Podstawy programowania na platformę Android**.

Aplikacja **ToDoList** służy do zarządzania zadaniami użytkownika. Projekt został wykonany w języku Kotlin z użyciem Jetpack Compose.

## Technologie

- Kotlin
- Jetpack Compose
- Room Database
- Retrofit REST API
- Google Maps
- Navigation Compose

## Główne funkcje

- ekran startowy Splash Screen,
- logowanie i rejestracja użytkownika,
- dodawanie zadań,
- oznaczanie zadań jako wykonane,
- usuwanie zadań,
- zapis danych lokalnie w Room,
- pobieranie danych z API przez Retrofit,
- nawigacja Bottom Navigation i Navigation Drawer,
- mapa Google Maps z markerem,
- intencje systemowe, np. telefon, przeglądarka, mapy.
